● Los agentes de código trabajan horas sin supervisión (Fable 5, Devin, Codex)
● MCP se consolida como el estándar para conectar agentes con todo
● Los open weights chinos (DeepSeek V4, Kimi, Qwen, GLM) dominan la gama abierta
● Agentes gestionados en la nube: el proveedor ejecuta el loop (Managed Agents, Codex cloud)
● Guerra de precios: salida desde $2.5/1M (Grok) y contexto de 2M tokens (Gemini)
