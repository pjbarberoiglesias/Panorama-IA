### 💻 CAPA 3 — IDEs con IA & Agentes de Código
> *Integrados en el editor. Escriben, depuran y despliegan software.*

| Herramienta | Empresa | Integración |
|---|---|---|
| ⭐ **Antigravity** | Google DeepMind | IDE nativo (este) |
| **Cursor** | Anysphere | IDE propio |
| **Windsurf** | Codeium | IDE propio |
| **VS Code + GitHub Copilot** | Microsoft | Extensión |
| **Claude Code** | Anthropic | Terminal |
| **Codex** | OpenAI | Nube/GitHub |
| **Devin** | Cognition AI | Web/Nube |
| **Zed** | Zed Industries | IDE propio |

**Diferencia clave entre ellos:**
- `Cursor / Windsurf / Zed` → IDEs independientes con IA muy integrada
- `VS Code + Copilot` → IA como extensión del editor estándar
- `Antigravity` → Agente en IDE con herramientas, memoria y contexto completo del workspace
- `Claude Code / Codex` → Agentes de terminal/nube, sin IDE visual
- `Devin` → El más autónomo: trabaja solo durante horas

## 🎯 ¿Dónde encaja cada cosa?

```
PREGUNTA RÁPIDA → Asistente IA (ChatGPT, Claude.ai)
DESARROLLO DE SOFTWARE → IDE con IA (Antigravity, Cursor, Windsurf)  
TAREA COMPLEJA DEL MUNDO REAL → Agente General (Manus, OpenClaw)
AUTOMATIZAR CON TU CÓDIGO → Framework (LangChain, CrewAI)
PRIVACIDAD / SIN COSTE API → Self-hosted (Ollama, OpenClaude)
```
