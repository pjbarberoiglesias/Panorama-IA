# 🌐 Panorama Global de la IA — 2026

Proyecto de material didáctico para el **Master en IA Aplicada**.  
Un informe interactivo completo del ecosistema de Inteligencia Artificial en 2026.

## 📁 Estructura del proyecto

```
1.Panorama IA/
├── panorama_ia_2026.html       ← Informe interactivo principal (abrir en navegador)
├── README.md                   ← Este archivo
└── assets/
    ├── panorama_ia_2026_resumen.md          ← Resumen en Markdown (6 capas)
    ├── panorama_ia_2026_*.png               ← Infografía del panorama (PNG)
    ├── panorama_ia_infografia_*.webp        ← Infografía completa (WebP)
    └── panorama_ia_preview_*.webp           ← Preview del diagrama (WebP)
```

## 🚀 Cómo usar

1. Abrir `panorama_ia_2026.html` directamente en el navegador (doble clic)
2. Navegar por las 5 pestañas:
   - 🗺️ **Resumen** — Las 6 capas del ecosistema IA
   - 🧠 **Modelos LLM** — Benchmarks y características de 14+ modelos
   - 🛠️ **Herramientas** — 40+ herramientas categorizadas
   - 💰 **Precios** — Suscripciones y precios de API
   - 📊 **Matriz Comparativa** — Comparación interactiva lado a lado

## 🗺️ Las 6 Capas del Ecosistema IA

| Capa | Tipo | Ejemplos |
|------|------|---------|
| 1 | Modelos Fundacionales (LLMs) | GPT-4o, Claude 3.7, Gemini 2.5, Llama 3.3 |
| 2 | Asistentes IA (Reactivos) | ChatGPT, Claude.ai, Gemini, Copilot |
| 3 | IDEs con IA & Agentes de Código | Antigravity, Cursor, Windsurf, Claude Code |
| 4 | Agentes IA Generales (Autónomos) | Manus, OpenClaw, Operator |
| 5 | Frameworks Multi-Agente | LangChain, CrewAI, AutoGen |
| 6 | Infraestructura Self-Hosted | Ollama, LM Studio, LocalAI |
